# WeeeeTanks
Name: Weeee Tanks
Based on the Wii Play game "Tanks!", a 2d game with a variety of AI controlled opponents 
where the player must defeat all the enemies using mines and bouncing bullets to move on to the next level.


Run main.py file

no additional libraries

to shortcut to a specific level, press the level-1 number on the keyboard and hit start
(only works up to level 10)
Controls:
WASD for movement
mouse to aim, click to fire
p to pause
